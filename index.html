Daily Email privacy policy

Daily Email is an Email client for Google TV that uses the user's Email provider to view and retrieve mail.Daily Email only supports user login and view mail functionality. Gmail user data is requested through the Google API service to help users access:

  • Sign in to Google mail;
  • Read all Emails from Gmail;
  • View Email contacts.

    • In order to process Gmail user data requested through Google API Services, Daily Email will comply with both ‘Google API Services: User Data Policy’ and ‘Google APIs Terms of Service’. Sensitive user information is used only for the purpose of executing the application.

    Sensitive user information is only used to perform the basic functions of the application, i.e. to send and receive Emails, and Daily Email does not collect data and send it to the application developers or any third parties.

    Please note that the Google APIs are only used when adding a Google/Gmail account to Daily Email.
